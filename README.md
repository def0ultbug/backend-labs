# Backend Labs 🔬

A collection of my hands-on experiments and mini-projects with Python for learning backend development concepts, frameworks, and architectural patterns.

## 📚 What's Inside

This repository documents my journey learning:
- **FastAPI** - Modern Python web framework for building APIs
- **Backend Architecture** - System design & Architectures, scalability, and best practices
- **Design Patterns** - Reusable solutions to common software problems

Each folder represents a focused learning exercise with its own README and implementation.

## 🚀 Getting Started

Each project folder contains:
- `README.md` - Concept explanation and learning objectives
- `requirements.txt` or `pyproject.toml` - Dependencies
- Source code with comments explaining key concepts
- Tests (where applicable)


### Running a Project
```bash
# Navigate to a specific project
cd fastapi/01_fastapi_router

# Run the application
python main.py

```
